# finance-tracker

## Project Description
This project is a finance tracker application written in Python using the Tkinter graphical interface library.

The program allows users to record and manage their personal income and expenses.

## Features
Add income and expense transactions
Categorize spending (Food, Travel, Bills, etc.)
Add custom categories
Automatically save transactions to a CSV file
View weekly and monthly expense totals
Calculate current balance
Delete individual transactions
Clear all transactions
View detailed financial summaries

## How to Run the Program
1. Install Python 3
2. Download this repository
3. Run the program:

python finance_tracker.py

## File Structure

finance_tracker.py  → Main program  
alinas_finance.csv  → Stores saved transactions (created automatically)
